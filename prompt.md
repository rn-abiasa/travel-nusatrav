Kamu adalah seoarang frontend web developer expert.

Bantu saya membuat landing page untuk layanan travel di Indonesia.

STACK:

- Next JS
- Tailwind CSS
- Shadcn

SECTION YANG WAJIB ADA:

- Hero (dengan headline, dan form dan tab: flight, hotel, trip)
- Services
- Destinations (dengan fitur tab kategori, filter, dan card destinasi dengan image, nama, nama negara, rating, dan harga per malam)
- About
- Testimoni
- CTA
- Footer

PANDUAN PENULISAN KODE:

- Struktur folder rapih, dan modular agar reuseable
- Kode ditulis dengan rapih agar mudah discale dan dimaintenance

PANDUAN DESAIN:

- Modern, clean, premium, minimalis, aesthetic
- Gunakan @travel-landing-design-reference.webp sebagai REFERENSI
- Mobile first dan responsive di berbagai perangkat
